# Machine-Learning-Libraries
## Overview
This repository contains Experiment 17 on Machine Learning Libraries. The document explains the most commonly used Python libraries in Machine Learning, their features, applications, advantages, and basic examples.

## Topics Covered

- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow
- Keras
- PyTorch
- SciPy
- OpenCV

## Objectives
- Understand the role of libraries in Machine Learning.
- Learn data manipulation and preprocessing techniques.
- Explore data visualization tools.
- Study Machine Learning and Deep Learning frameworks.
- Understand computer vision and scientific computing libraries.

## Technologies Used
- Python
- Machine Learning Libraries
- Data Visualization Tools
- Deep Learning Frameworks

## Learning Outcomes
After completing this experiment, learners will be able to:
- Use NumPy and Pandas for data handling.
- Create visualizations using Matplotlib and Seaborn.
- Build Machine Learning models using Scikit-learn.
- Develop Deep Learning applications using TensorFlow, Keras, and PyTorch.
- Perform image processing tasks using OpenCV.
